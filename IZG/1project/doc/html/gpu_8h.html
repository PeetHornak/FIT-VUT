Detailed Description

This file contains structures and function declarations of GPU HW. Main structure is GPU that holds all components of the GPU. GPU is composed of list of buffers, list of vertex pullers and list of programs. GPU also contains framebuffer, active program and vertex puller and states. GPU executes commands that are stored in CommandQueue.
